# Memory Game Images

Ten folder zawiera obrazki do gry pamięciowej.

## Instrukcje:

1. Dodaj 15 różnych zdjęć (pary)
2. Nazwy plików powinny być: `1.jpg`, `2.jpg`, `3.jpg`, ..., `15.jpg`
3. Zalecany format: JPG lub PNG
4. Zalecany rozmiar: około 200x200px (lub proporcje kwadratowe)

## Aktualne zdjęcia:

- Brak - dodaj swoje zdjęcia!

## Podpowiedź:

Możesz użyć zdjęć:
- Wspólnych wspómnień
- Miejsc, które odwiedziliście razem
- Ukochanych zwierząt
- Ulubionych rzeczy osoby, która gra
- Świątecznych motywów

Gra automatycznie załaduje zdjęcia z tego folderu. Jeśli zdjęcia nie będą istniały, wyświetlą się placeholdery.
